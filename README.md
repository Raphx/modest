# Modest

Code with modesty.

No distraction, no addiction, one step closer to nirvana.

Inspired by [nofrils](https://github.com/robertmeta/nofrils).

## Usage

Modest uses only 16-color mode, built only for terminal vim.

Two color schemes are available:

**Modest Light**
